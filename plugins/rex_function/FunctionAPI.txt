1. API of Function plugin  (export to other plugins)
A. set parameters
- fn_obj.AddParams(param)              // param is a hash

B. call function
- fn_obj.ExecuteCommands(cmd)          // cmd is a string
- fn_obj.CallFn(name, args)            // args is a hash
- fn_obj.CreateJS(name, code_string)


2. API of internal function object  (using in JS)
A. set parameters, returns
- fn["param"]                          // a hash
- fn["ret"]                            // a hash
- fn["result"]                         // single value

B. call function
- fn["CallFn"](name, param0, param1...)
- fn["CreateJS"](name, code_string)
- fn["_CallFn"](name, args)            // args is a hash
